# Proyecto Frontend - React + TypeScript + Vite

Este proyecto es una aplicación frontend construida con [Vite](https://vitejs.dev/) utilizando [React](https://react.dev/) y [TypeScript](https://www.typescriptlang.org/).

## Requisitos

- [Node.js](https://nodejs.org/) v16 o superior
- [npm](https://www.npmjs.com/)

## Instalación

1. Clona este repositorio
```
git clone https://gitlab.com/pisw2025-estudio-psicologico/pisw2025_estudiopsicologico/frontend_project_centropsicologico_pisw2025.git
```

2. Instala las dependencias
```
npm install
```

## Levantar el proyecto localmente
```
npm run dev
```
Esto abrirá la aplicación en tu navegador en http://localhost:5173. Asegúrate de tener el este puerto disponible, o Vite elegirá uno alternativo.
